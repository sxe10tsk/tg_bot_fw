Документация Telegram-бота
1. Назначение
Бот предназначен для помощи в изучении английских слов. Пользователь имеет персональный словарь,
куда может добавлять и удалять слова. Бот показывает случайное слово из личного словаря пользователя и
предлагает выбрать правильный перевод из четырёх вариантов где три ложные, а один верный.

2. Команды и кнопки
Основные команды
/start для начала работы.
При первом сообщении боту создаётся новый "пользователь", который добавляется в таблицу после чего ему добавляются
слова по умолчанию. Далее показывается первая карточка для перевода.

Во время общения с ботом доступны следующие кнопки:
    Варианты перевода – четыре кнопки с английскими словами (одно правильное, три случайных из вашего словаря).
    Дальше – переход к следующей карточке.
    Добавить слово – позволяет добавить пользователю английское слово и его перевод.
    Удалить слово – позволяет удалить текущее слово из карточки из словаря пользователя.

3. Требования для разработчика

Python 3.8+
PostgreSQL

Библиотеки: sqlalchemy, psycopg2, pyTelegramBotAPI
Так же необходимо указать свой token_bot и настройки подключения к БД.
